# News-Analyser-Website-
A dynamic web application that scrapes headlines across categories such as Sports,Finance,Technology,Business,World,Science and Entertainment. It provides real-time updates, integrates sentiment analysis to classify articles into positive, negative, or neutral, and visualizes insights with interactive charts for better understanding of news trends.

A simple news scraping and sentiment analysis website built using Python, Flask, BeautifulSoup, and Chart.js.

## Features
- Scrapes latest news headlines
- Category-wise news pages
- Sentiment analysis (Positive/Negative/Neutral)
- Simple and fast UI

## Tech Stack
- **Backend:** Flask, BeautifulSoup, Requests  
- **Frontend:** HTML, CSS, JavaScript, Chart.js  
- **Deployment:** Netlify + PythonAnywhere
